Movie API Project

CF Full-Stack Immersion Achievement 2

Objective:
To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Movies

Request Method URL Response
Return list of all movies. GET Movies/ res.json Movies list
Return data about specific movie GET movies/:movie res.json movie obj
Return data about a genre GET Movies/:genre res.json genre list
Return data about a director GET movies/:director res.json director obj
